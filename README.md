# GREEN-API

Проект представляет собой Single Page Application (SPA) для тестирования базовых методов GREEN-API, разработанный в рамках тестового задания.

## 🛠 Стек технологий
- HTML5 (Семантическая верстка)
- CSS3 (Custom Properties, Grid, Flexbox, Адаптивность < 768px)
- Vanilla JavaScript (ES6+, Async/Await)

## 🏗 Архитектурные особенности
- **Слабая связность (Loosely Coupled):** Четкое разделение приложения на UI-слой, Бизнес-логику (Валидация/Санитизация) и изолированный API-клиент.
- **Принцип DRY:** Все запросы к API инкапсулированы в единой функции `apiRequest`.
- **Уровень надежности (L2 Support):** Реализован механизм таймаута запросов (15с) через `AbortController`, детальный перехват сетевых сбоев (CORS, DNS) и разбор дефектов на стороне сервера (HTTP Status).
